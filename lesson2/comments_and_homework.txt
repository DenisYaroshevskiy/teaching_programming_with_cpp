Напоминаю - при создании нового проекта в VisualStusio выбирай консольное
приложение Win32.

1) Переписать цикл:
   for (int i = 0; i < 10; ++i) {
     cout << i << " ";
   }
   с помощью while.

2) Для всех двухзначных чисел, если число делится на 2 - foo, на 4 - bar.
    - написать как foo_bar
    - написать как foo bar, но проверять делимость на 4,
      только если делится на 2
    - печатать только foo или только bar (если число делиться на
                                          4 - не писать foo).
       * с else if
       * с continue

2) (будет очень здорово если сможешь) Попробовать написать вычисление n-ого числа
   фибоначчи (http://cyclowiki.org/wiki/Числа_Фибоначчи). Нужно будет сохранять
   результаты предыдущие результаты в переменных. Число фибоначчи, которое
   считаешь, можешь просто вбить в код или считать с ввода.
